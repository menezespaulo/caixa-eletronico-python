# Caixa Eletrônico 🏦

Projeto em Python que simula o sistema de um caixa eletrônico pelo terminal, desenvolvido para colocar em prática conceitos básicos e intermediários de lógica de programação.

O programa conta com uma tela inicial de login (validação de conta e senha) e, após a autenticação, libera um menu interativo para o usuário navegar pelas operações financeiras.

### 🧠 O que apliquei nesse código:
* **Estruturas Condicionais (`if`, `elif`, `else`):** Utilizadas tanto na validação das credenciais de acesso quanto no direcionamento das opções do menu.
* **Laços de Repetição (`while True`):** Aplicados para manter o usuário no sistema até que ele digite as informações corretas de login ou decida encerrar a sessão na opção de sair.
* **Manipulação e Formatação de Variáveis:** Uso de f-strings para exibir valores monetários formatados corretamente com duas casas decimais (`.2f`).
* **Tratamento de Entradas:** Menu preparado para lidar com opções inválidas digitadas pelo usuário sem quebrar a execução do programa.
